TO USE pyinstaller TO CREATE AN EXECUTABLE FILE FROM A GIVEN PYTHON SCRIPT, USE THE FOLLOWING COMMAND:

pyinstaller --onefile your_py_script.py

TO INSTALL THIS PROGRAM, MAKE SURE YOU'RE IN A PYTHON VIRTUAL ENVIRONMENT AND USE:

pip install pyinstaller 

TO GET THE PROGRAM

IF YOU AREN'T IN A VIRTUAL ENVIRONMENT, BUILD ONE USING:

FOR LINUX:
python3 -m venv venv 

FOR WINDOWS:
py -m venv venv
